# 配置参考

## 全局配置 `[global]`

| 字段 | 默认值 | 说明 |
|------|--------|------|
| `worker_threads` | 0（= CPU 核心数）| Worker 线程数 |
| `worker_connections` | 51200 | 最大并发连接数 |
| `max_connections` | 0（不限）| 全局连接上限，超出返回 503 |
| `keepalive_timeout` | 60 | Keep-Alive 超时（秒） |
| `client_max_body_size` | 50 | 最大请求体（MB） |
| `client_header_buffer_size` | 32 | 请求头缓冲区（KB） |
| `gzip` | false | 全局 gzip 压缩 |
| `gzip_min_length` | 1 | 最小压缩大小（KB） |
| `gzip_comp_level` | 5 | 压缩等级 1-9 |
| `admin_listen` | — | 管理 API 监听地址（建议绑定 127.0.0.1） |
| `admin_token` | — | 管理 API Bearer Token |
| `log_level` | — | tracing 日志过滤器（如 `sweety_lib=info`） |

## 站点配置 `[[sites]]`

| 字段 | 说明 |
|------|------|
| `name` | 站点唯一标识 |
| `server_name` | 域名列表（精确 / `*.wildcard` / `fallback = true` 兜底） |
| `listen` | HTTP 端口列表 |
| `listen_tls` | HTTPS 端口列表 |
| `root` | 站点根目录 |
| `index` | 索引文件列表 |
| `access_log` | 访问日志文件路径 |
| `access_log_format` | `combined` / `json` / 自定义模板 |
| `force_https` | HTTP 自动 301 跳转 HTTPS |
| `fallback` | 无 SNI 匹配时的兜底站点 |
| `gzip` | 站点级覆盖全局 gzip |
| `websocket` | 允许 WebSocket 升级 |
| `error_pages` | `{404 = "404.html"}` 自定义错误页 |

## TLS 配置 `[sites.tls]`

| 字段 | 说明 |
|------|------|
| `acme` | 启用 ACME 自动证书 |
| `acme_email` | ACME 账号邮箱 |
| `acme_provider` | `letsencrypt` / `zerossl` / `buypass` |
| `acme_challenge` | `http01` / `dns01` |
| `acme_renew_days_before` | 提前 N 天续期 |
| `certs[]` | `{cert = "...", key = "..."}` 静态证书（支持多证书 SNI 自动选最优） |
| `min_version` / `max_version` | `tls1.2` / `tls1.3` |

## 上游配置 `[[sites.upstreams]]`

| 字段 | 默认值 | 说明 |
|------|--------|------|
| `name` | — | 上游组名 |
| `strategy` | `round_robin` | `round_robin` / `weighted` / `least_conn` / `ip_hash` |
| `connect_timeout` | 10 | 连接上游超时（秒） |
| `read_timeout` | 60 | 读取响应超时（秒） |
| `write_timeout` | 60 | 写入请求超时（秒） |
| `retry` | 0 | 失败重试次数 |
| `retry_timeout` | 0 | 重试前等待（秒） |
| `keepalive` | 32 | 连接池最大空闲连接数 |
| `keepalive_requests` | 1000 | 单连接最大请求数 |
| `keepalive_time` | 60 | 连接最大存活时间（秒） |

### 断路器 `[sites.upstreams.circuit_breaker]`

| 字段 | 默认值 | 说明 |
|------|--------|------|
| `max_failures` | 5 | 滑动窗口内失败次数触发开路 |
| `window_secs` | 60 | 滑动窗口（秒） |
| `fail_timeout` | 30 | 开路后半开探测间隔（秒） |

### 节点 `[[sites.upstreams.nodes]]`

| 字段 | 说明 |
|------|------|
| `addr` | `host:port` |
| `weight` | 负载均衡权重 |
| `tls` | 启用到上游的 TLS |
| `tls_sni` | TLS SNI 主机名 |
| `tls_insecure` | 跳过上游证书验证（内网自签证书） |
| `upstream_host` | 覆盖发往上游的 Host 头 |

## Location 配置 `[[sites.locations]]`

| 字段 | 说明 |
|------|------|
| `path` | 匹配规则（`/` 前缀 / `= /exact` 精确 / `^~ /prefix` 前缀优先 / `~ regex` 正则） |
| `handler` | `static` / `reverse_proxy` / `fastcgi` / `grpc` / `websocket` / `plugin:xxx` |
| `upstream` | 上游组名（reverse_proxy / grpc 使用） |
| `root` | 覆盖站点根目录 |
| `try_files` | `["$uri", "$uri/", "/index.php"]` |
| `return_code` | 直接返回 HTTP 状态码 |
| `return_url` | `"301 https://..."` 重定向 |
| `return_body` | 直接返回文本内容 |
| `return_content_type` | return_body 的 Content-Type |
| `limit_conn` | 该 location 最大并发连接数 |
| `proxy_buffering` | 缓冲上游响应（默认 true） |
| `auth_request` | 子请求鉴权 URL |
| `auth_failure_status` | 鉴权失败返回状态码（默认 401） |
| `proxy_set_headers[]` | `{name, value}` 覆盖发往上游的请求头，支持变量 |
| `add_headers[]` | `{name, value}` 注入响应头 |
| `sub_filter[]` | `{pattern, replacement}` 响应体内容替换 |
| `cache_rules[]` | `{pattern, cache_control}` 按路径覆盖 Cache-Control |
| `strip_cookie_secure` | 去掉 Set-Cookie 的 Secure 标志 |
| `proxy_cookie_domain` | 替换 Set-Cookie 的 Domain |
| `proxy_redirect_from/to` | 重写 Location 响应头 |

## proxy_set_headers 变量

`$remote_addr` `$host` `$scheme` `$request_uri`

## 访问日志模板变量

| 变量 | 说明 |
|------|------|
| `$remote_addr` | 客户端 IP |
| `$method` | HTTP 方法 |
| `$uri` | 请求 URI |
| `$status` | 响应状态码 |
| `$bytes_sent` | 响应字节数 |
| `$http_referer` | Referer 头 |
| `$http_user_agent` | User-Agent 头 |
| `$duration_ms` | 请求耗时（ms） |
| `$time_local` | 本地时间 |
| `$site` | 站点名 |
